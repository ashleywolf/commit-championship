# Commit Championship

Enter a GitHub username. Get announced like a wrestling champion.

**[Play it](https://ashleywolf.github.io/commit-championship/)**

Your GitHub stats become a wrestling championship poster. Repos, stars, and account age determine your tier -- from Newcomer Showcase (bronze) to Undisputed Champion (cyan neon). Pyrotechnics fire. Lens flares sweep. The announcer calls out your stats one by one.

5 championship tiers. Belt rendered on canvas. Downloadable as a poster.

## How it works

- GitHub API for profile stats (repos, stars, account age)
- 5 championship tiers with distinct color palettes
- Pyrotechnic particle effects scaled by tier
- Animated belt with shine effect
- Announcer callout reveals with staggered timing
- Web Audio API, Canvas 2D rendering
- Shareable poster export (save/copy/share)
- Single HTML file, no build tools
